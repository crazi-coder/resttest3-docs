REST Testing Tool (resttest3)
===============================

What Is Resttest3?
----------------------

-  A REST testing and API microbenchmarking tool
-  Tests are defined in basic YAML or JSON config files, no code needed
-  Minimal dependencies making it easy to deploy
   on-server for smoketests/healthchecks
-  Supports generate mechanisms
   to create full test scenarios with workflow
-  Support  extension

Getting Started
---------------
- If you are new to restetest3, you can get started by following the `First Steps with Resttest3 <quickstart.html#simple-testcase>`__
